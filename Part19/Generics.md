### Generics

- Gen in ts allows to u to create reusable components or functions that can work with multiple data types.


#### Syntax

- Deneratic is defined by inside the <>.
- we just need to pass a variable inside the <> of any type.
- <> Has the reference of data type which is used while making function call.

```

const function_Name = <Generic_Name>( parameter : Generic_Name ) : GenericName => {
    return Type
}

function_Name<type annotations>()
```
